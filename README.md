## üì¶ Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Building for Production

Build the application for production:
```bash
npm run build
```

### Running Tests

Run the unit tests:
```bash
npm test
```
